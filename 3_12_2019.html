End of year letter to the creative computer club.

December 3, 2019

Hello!

Congratulations on completing a year of the ‘fantabulous’ creative computer club. I hope you’ll keep learning about things that interest you and you stay curious about solving fun problems with computers.

Here are some things that we have covered this year, which might be interesting:

  • Python programming,
  • HTML + CSS,
  • Scratch,
  • computational thinking and applied problem solving,
  • soft skills – communicating, how to create/play-well together.

All of the tools that we have used this year are ‘Free and Open source software’ – if you search for the tools you will be able to find them and download them for yourself. You could even ‘dual-boot’ a Linux distribution, if you wanted to get something that looked like Raspbian the Raspberry Pi OS, you could try Ubuntu Mate.

So for now, have fun for the summertime – don’t forget your sunblock!
Stay curious, keep learning. Be good!

Sam
