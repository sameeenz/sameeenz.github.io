Adding EXIF metadata to digitised photos with Linux CLI.

March 23, 2019

Today I got a few films from high school digitised, after they came off the scanner they didn’t have any metadata relating to the date in which they were actually taken.

I wanted to add a bit of metadata so they would be sensibly arranged in my photo content management system – I had to guess a bit. Within the nearest year was good enough.

I found that a really nice tool exists for this task called ‘exiftool’ created by Phil Harvey. Thanks Phil, cracker tool. On Ubuntu it can be installed by:

sudo apt-get install exiftool
